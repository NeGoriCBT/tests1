Шкала Гамильтона для оценки депрессии (HDRS)

Hamilton Depression Rating Scale, 17 пунктов

Важно. Тест не ставит диагноз и не является основанием для самолечения. Результаты носят информационный характер и не заменяют очную консультацию врача или психолога.

Конфиденциальность. Ответы и личные данные никуда не отправляются и не сохраняются на сервере: расчёт выполняется только в вашем браузере. Файл Word при скачивании создаётся у вас на устройстве. После закрытия страницы введённые ответы в браузере не хранятся.

Инструкция

Оригинальная шкала предназначена для оценки специалистом. В данной форме каждый пункт можно использовать для самооценки по описанию: внимательно прочитайте формулировки и отметьте один вариант в каждом номере, который лучше соответствует вашему состоянию за последние дни (в том числе сегодня).

Результат

Суммарный балл (0–52) —
